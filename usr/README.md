# Android Games Hub

A responsive Flutter application that displays a catalog of Android games. Built with a dark theme and modern Material 3 design principles, this app scales gracefully across mobile, tablet, and desktop screens.

## Features
- **Responsive Grid:** Automatically adjusts the number of columns based on screen width (from 2 columns on mobile to 5 on large desktops).
- **Game Cards:** Displays game artwork, title, genre, and rating.
- **Dark Mode UI:** Optimized for gaming aesthetics with a sleek dark background and vibrant accents.
- **Cross-Platform:** Runs seamlessly on iOS, Android, Web, Desktop, and embedded previews.

## Tech Stack
- Flutter (Dart)
- Material 3 Design
- Custom responsive LayoutBuilder constraints

## How to Run
1. Ensure you have Flutter installed.
2. Clone this repository and run `flutter pub get`.
3. Start the application with `flutter run`.
